# ITZFIZZ — Scroll-Driven Hero Section Animation

A premium scroll-driven hero animation built using GSAP and modern frontend technologies. This project focuses on delivering smooth motion, depth-based transformations, and a cinematic user experience inspired by high-end product landing pages.

---

## 🚀 Highlights

- Smooth scroll-synced animation using GSAP ScrollTrigger  
- 3D-like car motion using scale, rotation, and depth effects  
- Clean and modern UI with performance-first approach  
- Designed to replicate real-world product landing experiences  

---

## ✨ Features

### 🎯 Scroll-Driven Animation
- Car smoothly scales forward as the user scrolls  
- Headline fades and moves backward  
- Stats drift out with scroll progression  
- Fully controlled using GSAP ScrollTrigger  

---

### 🎬 Load Animations
- Character-by-character headline reveal (staggered animation)  
- Smooth car entrance animation  
- Stats appear sequentially with delay  

---

### 🎨 Premium UI Design
- Dark theme with red accent palette  
- Modern typography using Bebas Neue  
- Grain overlay for cinematic feel  
- Subtle grid texture for depth  

---

### ⚡ Performance Optimized
- Uses only **transform** and **opacity** for animations  
- No layout shifts or reflows  
- GSAP `scrub: 1.4` for smooth scroll synchronization  

---

### 📱 Responsive Design
- Works seamlessly on desktop and mobile  
- Optimized animation intensity for smaller screens  

---

## 🛠️ Tech Stack

- HTML  
- CSS / Tailwind CSS  
- JavaScript  
- GSAP (ScrollTrigger)  
- Next.js / React
